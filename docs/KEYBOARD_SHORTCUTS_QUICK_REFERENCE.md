# RTX Market Watch - Keyboard Shortcuts Quick Reference

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| **F9** | Open New Order Dialog |
| **F10** | Open Popup Prices |
| **Alt+B** | Open Depth of Market |
| **Ctrl+U** | Open Symbols Dialog |
| **Escape** | Close Modal/Menu |

## Context Menu Navigation

### Opening the Menu
- **Right-click** on any symbol to open context menu

### Navigating Items
| Key | Action |
|-----|--------|
| **↓** (Arrow Down) | Next item |
| **↑** (Arrow Up) | Previous item |
| **Enter** | Execute action |
| **Space** | Execute action |
| **Escape** | Close menu |

### Submenu Navigation
| Key | Action |
|-----|--------|
| **→** (Arrow Right) | Open submenu |
| **←** (Arrow Left) | Close submenu |
| **Escape** | Close submenu (then menu) |

## Tips

1. **Focus Indicator**: Blue highlight shows current selection
2. **Wrap Around**: Arrow keys loop from last to first item
3. **Skip Logic**: Auto-skips dividers and disabled items
4. **Mouse Still Works**: All mouse interactions preserved

## Accessibility

- Full screen reader support
- WCAG 2.1 compliant
- Focus trap prevents accidental tab-out
- Clear visual focus indicators
