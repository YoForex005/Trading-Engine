  - DisplayName string
  - BaseCurrency, QuoteCurrency string

  - ContractSize decimal.Decimal (units per lot/contract)
  - MarginRate decimal.Decimal (required margin percentage, e.g., 0.10 for 10%)

  - TradingHoursID string (references trading hours definition)
  - LPSource string (which LP provides this symbol)
  - MinLotSize, MaxLotSize, LotStep decimal.Decimal
  - PipValue decimal.Decimal
  - Enabled bool
  - CreatedAt, UpdatedAt time.Time

Use govalues/decimal for all financial values (not float64). Follow the pattern from Phase 3 decision.

Export functions:
- NewSymbolSpec(...) *SymbolSpec - constructor with validation
- (s *SymbolSpec) Validate() error - validates all required fields
- (s *SymbolSpec) CalculateMargin(volume decimal.Decimal, price decimal.Decimal) decimal.Decimal - returns required margin

<task type="auto">
  <name>Task 2: Add symbol metadata database schema</name>
  <files>backend/database/schema.sql, backend/database/repositories/symbol_repository.go</files>
  <action>
Extend database schema with symbol_metadata table:

```sql
CREATE TABLE IF NOT EXISTS symbol_metadata (
    symbol VARCHAR(50) PRIMARY KEY,
    asset_class VARCHAR(20) NOT NULL,
    display_name VARCHAR(100) NOT NULL,
    base_currency VARCHAR(10),
    quote_currency VARCHAR(10),
    tick_size DECIMAL(30, 10) NOT NULL,
    contract_size DECIMAL(30, 10) NOT NULL,
    margin_rate DECIMAL(10, 6) NOT NULL,
    swap_rate_long DECIMAL(10, 6),
    swap_rate_short DECIMAL(10, 6),
    trading_hours_id VARCHAR(50),
    lp_source VARCHAR(50) NOT NULL,
    min_lot_size DECIMAL(30, 10),
    max_lot_size DECIMAL(30, 10),
    lot_step DECIMAL(30, 10),
    pip_value DECIMAL(30, 10),
    enabled BOOLEAN DEFAULT true,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE INDEX idx_symbol_asset_class ON symbol_metadata(asset_class);
CREATE INDEX idx_symbol_lp_source ON symbol_metadata(lp_source);
CREATE INDEX idx_symbol_enabled ON symbol_metadata(enabled);
```
