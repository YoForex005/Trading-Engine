**Implement LPAdapter interface methods:**
- ID() returns "interactivebrokers"
- Name() returns "Interactive Brokers"
- Type() returns "REST/WebSocket"
- Connect() - validates credentials with /v1/api/iserver/auth/status
- GetSymbols() - fetches from /v1/api/trsrv/secdef, maps to SymbolSpec[]
- GetQuotes() - fetches from /v1/api/md/snapshot
- Subscribe/Unsubscribe - implement quote streaming

**Asset Class Mapping (from RESEARCH.md):**
Map IB secType to unified AssetClass:
- "STK" → metadata.AssetClassStock
- "CASH" → metadata.AssetClassForex
- "CMDTY" → metadata.AssetClassCommodity
- "IND" → metadata.AssetClassIndex
- "CFD" → metadata.AssetClassCFD

**Symbol Spec Mapping:**
From IB security definition response, populate:
- Symbol: IB symbol (e.g., "AAPL")
- DisplayName: IB description
- TickSize: IB minTick value
- ContractSize: IB multiplier (or 1.0 for stocks)
- MarginRate: from IB margin requirements (or default 0.25 for stocks)
- TradingHoursID: map to "NYSE" for US stocks, etc.
- LPSource: "InteractiveBrokers"

Use decimal.NewFromFloat() for conversions from IB API floats.
Handle authentication with Bearer token in Authorization header.

Reference OANDA and Binance adapters for error handling patterns.
