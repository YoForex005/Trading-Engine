===============================================================================
HTTP API ENDPOINTS
===============================================================================

User Management:
  POST   /admin/fix/provision              - Provision FIX access
  POST   /admin/fix/revoke                 - Revoke user access
  POST   /admin/fix/suspend                - Suspend user
  POST   /admin/fix/reactivate             - Reactivate suspended user
  POST   /admin/fix/regenerate-password    - Generate new password

Credentials:
  GET    /admin/fix/credentials            - List all credentials
  GET    /admin/fix/credentials/user       - Get user credentials

Sessions:
  GET    /admin/fix/sessions               - List all active sessions
  GET    /admin/fix/sessions/user          - Get user sessions
  POST   /admin/fix/sessions/kill          - Kill session

Rate Limiting:
  GET    /admin/fix/rate-limits            - List all rate limit states
  POST   /admin/fix/rate-limits/update     - Update user tier
  POST   /admin/fix/rate-limits/reset-violations - Reset violations
  GET    /admin/fix/rate-limits/tiers      - List available tiers

Rules:
  GET    /admin/fix/rules                  - List all rules
  POST   /admin/fix/rules/add              - Add rule
  POST   /admin/fix/rules/remove           - Remove rule
  POST   /admin/fix/rules/enable           - Enable rule
  POST   /admin/fix/rules/disable          - Disable rule

Statistics:
  GET    /admin/fix/stats                  - System statistics
