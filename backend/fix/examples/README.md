# Examples

This directory contains sample code demonstrating how to use the FIX gateway and libraries.

## Planned Examples

### Basic Connection
```
examples/basic-connection/
└── main.go
```
Simple example showing how to connect to YoFx FIX server.

### Market Data Subscription
```
examples/market-data/
└── main.go
```
Subscribe to real-time market data feeds.

### Order Placement
```
examples/order-placement/
└── main.go
```
Place, modify, and cancel orders.

### Complete Trading Strategy
```
examples/custom-strategy/
└── main.go
```
Full example implementing a simple trading strategy.

## Usage

Each example is a standalone Go program that can be run directly:

```bash
cd examples/basic-connection
go run main.go
```
